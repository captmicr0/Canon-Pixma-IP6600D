  • Mac OS X information

    According to TVUONGPHAM , once the printer's destination was set to support CDr the "CDR Tray-B" option the option appeared with his non-US drivers. Also he tested the US drivers while the printer's destination was set to support CDr and the option remained. This suggests that no special considerations are required for the print driver. Once the printer is set to support CDr then "CD-R Tray B" should appear in the printer utility. See Fig 4-1 And link to CD printing software.

    Mac users may enjoy downloading Canon's CD-LabelPrint for testing. They may also ignore Section 6.

    Yonatron is the first user sited with an iP6000 and an Airport Express base station.

      1. Turn printer off

      2. Run Airport Admin to see if it's listed

      3. If listed Delete from OS X printer list via Preference Pane

      4. Turn printer on

      5. Run Airport Admin to see that it was recognized

      6. Added to list via Preference Pane

      7. Select “PRINT” from an application to see if CD-R Tray **

      Fig 4-1

  • Windows information

    Windows users who installed the drivers without the printer's destination set to CDr support will not see "Printable disc" nor "Disc Tray" as an option as the drivers check for this support while installing the printer and add this to the registry. If you change the registry the option will appear regardless of whether the printer's destination is set to support CD-R or not.

    If you didn't set the destination before installing the drivers, or you otherwise don't see "Printable Disc" as an option for media type, see Section 6.

    Windows users may enjoy downloading Canon's CD-LabelPrint for testing. See links.
