[6] Windows Registry Key

Mac users may ignore this section and proceed to Section[7].

The last thing preventing CD-r printing on the Pixma Series is a registry key entry “CnmFSI_CDRSWITCH”. If you have a non CD-r support, when installing the driver this value will be set to 0, where if you install the driver with a printer's destination set to CD-r support this value will be one. If your printer is listed in the table below all you need to do is download the listed registry entry and double click to run it. Windows will ask you if you want to add the information to your registry and click yes. This will enable the feature in the driver. You may test to see if it's enabled by trying to print something and seeing if “Printable Disc (recommended)” is listed as a Media Type (fig 6-1) .

If your printer isn't listed in the table then you must find run regedit your self. To do so press <START> <RUN> and type in the world “regedit”. See fig 6-2.

In regedit the most simple way to find the key you need is to go to the [Edit] pull down menu and select Find and look for the word “CnmFSI_CDRSWITCH” and select find next. See fig 6-3. This will take while so be patient.


fig 6-3

Once found it will display in the window right highlighted. Just double click on the “CnmFSI_CDRSWITCH” text and change the value from 0 (no CD/DVD printing support) to 1 (CD/DVD printing support) and press [OK}. If you have more than one printer installed you may wish to press F3 to see if there is another entry of “CnmFSI_CDRSWITCH”.



If you prefer locating this key manually just navigate to

HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Print\Printers\Canon PIXMA iPxxxx\PrinterDriverData
